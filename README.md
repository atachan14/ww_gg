# ww_gg

人狼ゲームの公開情報から、各行動に対する期待勝率を計算するツール。

## 現在のドキュメント
- `docs/requirements.md`: プロジェクト概要、画面構成、用語、計算方針
- `docs/spec_v1.md`: v1 の実装範囲と完成目標
- `app/README.md`: 新実装の作業場所
- `archive/legacy_proto/README.md`: 旧試作コードの扱い

## ディレクトリ構成
```text

  docs/

```
